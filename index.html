矩生成函数(Moment Generating Function, MGF)

作用：它将随机变量的分布和统计特性表示为一个函数的形式，用来计算随机变量的各阶矩和其他统计特性

定义：随机变量\(X\)的概率密度函数为 \(f(X)\) 。一个实值函数 \(M:\mathbb{R} \rightarrow \mathbb{R}\)定义为

如果这个期望值在区间 \(−h<t<h\)内的所有\(t\)上存在(其中 h>0 )，则称 M(t) 为矩生成函数。

通常来说，不是所有的随机变量都有矩生成函数，但是，对于一个随机变量而言，如果其矩生成函数存在，那么它就是唯一的。

如果随机变量X是离散的 \[ M(t)=\sum_{x\in R_{X}}e^{t x}f(x) \] 如果随机变量X是连续的 \[ M(t)=\int_{-\infty}^{\infty}e^{t x}f(x)d x \]

性质1：矩生成函数的n阶矩在t=0出的取值就是随机变量的n阶原点矩。

例子：

指数分布的概率密度函数（PDF）为：\(f(x) = \lambda e^{-\lambda x}, \quad x \geq 0, \lambda > 0\) \[ \begin{align} M_X(t) &= \int_0^{\infty} e^{tx} \lambda e^{-\lambda x} dx\\ &= \lambda \int_0^{\infty} e^{(t - \lambda)x} dx \end{align} \] 当 \(t < \lambda\) 时，积分收敛，计算结果如下： \[ M_X(t) = \lambda \frac{1}{\lambda - t}, \quad t < \lambda \] 计算均值（一阶矩）

均值由矩生成函数的一阶导数在 \(t=0\) 处给出： \[ \mathbb{E}[X] = M'_X(0) \] 首先求导： \[ M'_X(t) = \frac{\lambda}{(\lambda - t)^2} \] 令 \(t=0\)： \[ \mathbb{E}[X] = M'_X(0) = \frac{\lambda}{\lambda^2} = \frac{1}{\lambda} \] 计算方差（二阶矩） \[ \text{Var}(X) = \mathbb{E}[X^2] - (\mathbb{E}[X])^2 \] 计算二阶矩： \[ \mathbb{E}[X^2] = M''_X(0) \] 求二阶导数： \[ M''_X(t) = \frac{2\lambda}{(\lambda - t)^3} \] 最终计算方差： \[ \text{Var}(X) = \frac{2}{\lambda^2} - \left(\frac{1}{\lambda}\right)^2 = \frac{2}{\lambda^2} - \frac{1}{\lambda^2} = \frac{1}{\lambda^2} \] 性质2：如果随机变量 \(X,Y\) 独立， \(a,b,c\in \mathbb{R}\)为常数，则 \[ M_{aX+bY+c}(t)=e^{ct}M_X(at)M_Y(bt) \] 证明： \[ \begin{align} M_{aX+bY+c}(t)&=E(e^{(aX+bY+c)t})\\&=e^{ct}E(e^{(aX+bY)t})\\&=e^{ct}E(e^{(aX)t})E(e^{(bY)t})\\&=e^{ct}M_X(at)M_Y(bt) \end{align} \] 性质3：如果两个随机变量 X 和 Y 具有相同的矩生成函数(MGF)在某个区间内收敛，则它们具有相同的分布： \[ M_X(t) = M_Y(t) \quad \Rightarrow \quad f(X) \sim f(Y) \] 例子：

对于随机变量 \(X\) ，我们知道 \[ \begin{align} \nonumber M_X(t)=\frac{2}{2-t}, \textrm{ for }t \in (-2,2). \end{align} \] 上述 MGF 是指数随机变量的 MGF，其中 \(\lambda=2\)。因此，我们得出结论\(X∼Exponential(2)\)
